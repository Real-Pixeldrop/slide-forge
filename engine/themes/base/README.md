# SlideForge Base Theme

Thème adaptatif qui s'ajuste automatiquement au brand kit fourni.

## Variables CSS

Le thème utilise des CSS custom properties injectées dynamiquement :

| Variable | Description | Défaut |
|----------|-------------|--------|
| `--brand-primary` | Couleur principale | `#1a1a2e` |
| `--brand-secondary` | Couleur secondaire | `#16213e` |
| `--brand-accent` | Couleur d'accent | `#0f3460` |
| `--brand-font` | Police du body | `Inter` |
| `--brand-font-heading` | Police des titres | `Inter` |

## Layouts disponibles

- **cover** : Slide de couverture (gradient primaire/secondaire)
- **section** : Transition de section (fond primaire)
- **default** : Contenu standard
- **two-cols** : Deux colonnes
- **image-right** : Texte + image à droite
- **image-left** : Image à gauche + texte
- **image** : Image plein écran
- **quote** : Citation
- **stats** : Chiffres clés (grille auto)
- **team** : Présentation d'équipe
- **bullets** : Liste stylée
- **end** : Slide de fin (gradient)
